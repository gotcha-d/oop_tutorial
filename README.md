## カプセル化
- 整合性を保つ範囲をクラスとして定義する
  - じゃんけん判定とその表示を一つにまとめる
- クラス単位で再利用する
- クライアントはメソッドを利用する（クラス内部に依存しない）
  - 変更が可能になる
- カプセル化は見ないふりをすること。
